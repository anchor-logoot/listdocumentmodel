addSpaceBefore

  • addSpaceBefore(s: number, rootUpdate?: undefined | ((np: AnchorLogootNode | undefined) => void)): void

  • Inherited from AnchorLogootNode.addSpaceBefore

    • Defined in node_modules/@anchor-logoot/dtst-js/lib/index.d.ts:134

    Applies an offset to the node's starting position. This will offset all nodes after this one efficiently. There's just one catch: This may not change the order of the TST. No errors will be thrown (or can, efficiently at least), so just make sure your code doesn't change the order. This will not mutate the TST unless the position of a node becomes equal to that of another node. In that case, the TST will be re-arranged so that the equal nodes form a linked list under the left side of the root equal node as required.

    Parameters

    • s: number

      The offset to apply to this node's start. This may be positive to add space before or negative to remove space before.

    • Optional rootUpdate: undefined | ((np: AnchorLogootNode | undefined) => void)

      A function that will be called if the TST root needs to be updated. A TypeError will be thrown if one is not provided.

    Returns void

  • Inherited from AnchorLogootNode.remove

    • Defined in node_modules/@anchor-logoot/dtst-js/lib/index.d.ts:118

    Removes this node from the TST. The node's absolute_value will be preserved, so it can be directly re-added to the TST.

    Parameters

    • Optional rootUpdate: undefined | ((np: AnchorLogootNode | undefined) => void)

      This function is called if the tst_root of the TST must be updated. Normally, it should be given the value of (node) => (tst.tst_root = node). If no value is provided, an error will be thrown if an update is attempted. For this reason, if you expect that no root update will be required for whatever reason, do not provide a value. This will ensure that an error is thrown and a potential bug is caught.

    Returns void
